laika.io.model

RenderedDocument 

case class RenderedDocument(path: Path, title: Option[SpanSequence], sections: Seq[SectionInfo], content: String, config: Config) extends RenderContent with DocumentNavigation with Product with Serializable

A single rendered document with the content as a plain string in the target format.

The title and section info are still represented as an AST, so they be used in any subsequent step that needs to produce navigation structures.
